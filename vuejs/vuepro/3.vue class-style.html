Vue class & style

vue.js中，class和style还可以通过其他方式来赋属性值。特别地，vue.js还不需要程序员给不同浏览器加前缀，vue.js会自动为他们加上前缀

class绑定

可以用v-bind:class="{'active':isActive,'red-bg':isRed}"来确定class值包含与否。

也可以用v-bind:class="[activeClass,bgColorClass]"这样一个数组将'active' 'red-bg'传给class。

style绑定

style同样可以通过v-bind:style="{fontSize:size,backgroundColor:bgColor}"来将size和bgColor赋给style。

或者直接v-bind:style="styleObject"绑定一个字典styleObject:{fontSize:'26px',backgroundColor:'pink',}，会更加清晰。

相应地，甚至可以通过v-bind:style="[styleObject1,styleObject2]"绑定一堆style的字典。
