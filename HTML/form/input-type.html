Input输入类型

input的输入类型参考W3school的Input-Type。

text, radio, submit类型见form，下面介绍type为其他类型会产生的不同输入类型。

<input type="password">定义密码字段：

User name:
Password:

<input type="checkbox">定义复选框。

复选框允许用户在有限数量的选项中选择零个或多个选项。

I have a bike
I have a car

<input type="button">定义按钮。

作用类似于<button>。

其他还有HTML5增加的多个新的输入类型。

注释：老式web浏览器不支持的输入类型，会被视为输入类型text。

<input type="number">用于应该包含数字值的输入字段。

还可以对数字进行限制，如max, min, maxlength, disabled, pattern等。

移动端可以直接唤出数字键盘。

Quantity (between 1 and 5):

<input type="date">用于应该包含日期的输入字段。

同样可以对日期进行限制，如min, max。

根据浏览器支持，日期选择器会出现在输入字段中。

Birthday:

<input type="month">允许用户选择月份和年份。

同样可以对月份进行限制，如min, max。

根据浏览器支持，日期选择器会出现在输入字段中。

Birthday (month):

<input type="week">允许用户选择周和年。

根据浏览器支持，日期选择器会出现在输入字段中。

Select a week:

<input type="time">允许用户选择时间（无时区）。

根据浏览器支持，时间选择器会出现在输入字段中。

Select a time:

<input type="datetime">允许用户选择日期和时间（有时区）。

根据浏览器支持，时间选择器会出现在输入字段中。

Birthday (date and time):

<input type="datetime-local">允许用户选择日期和时间（无时区）。

根据浏览器支持，时间选择器会出现在输入字段中。

Birthday (date and time):

<input type="color">用于应该包含颜色的输入字段。

根据浏览器支持，颜色选择器会出现在输入字段中。

Your favorite color:

<input type="range">用于应该包含一定范围内的值的输入字段。

根据浏览器支持，输入字段能够显示为滑块控件。

Stars (0-5):

<input type="email">用于应该包含电子邮件地址的输入字段。

根据浏览器支持，能够在提交时自动对电子邮件地址进行验证。

E-mail:

<input type="search">用于搜索字段（搜索字段的表现类似常规文本字段）。

Search Google:

<input type="tel">用于应该包含电话号码的输入字段。

Telephone:

<input type="url">用于应该包含URL地址的输入字段。

根据浏览器支持，能够在提交时自动验证url字段。
